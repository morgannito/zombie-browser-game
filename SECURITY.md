# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Please report security issues via email: **informatique@bonsecours66.com**

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

You will receive an acknowledgment within **48 hours** and a status update within **7 days**.

If the vulnerability is confirmed, a fix will be prioritized based on severity. You will be credited in the release notes unless you prefer to remain anonymous.

## Scope

This policy covers the zombie-browser-game server and client code hosted in this repository. Third-party dependencies are out of scope — please report those to their respective maintainers.
